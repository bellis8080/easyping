1. When agent focuses on message input box, Echo analyzes ticket context
2. Suggested response appears in Echo panel (collapsible sidebar)
3. Agent can click "Use This Response" to copy Echo's draft into reply box
4. Agent can edit Echo's suggestion before sending
5. Suggested response is editable within Echo panel before using it
6. "Generate Another" button requests alternative suggestion
7. Suggestions based on ticket history, category, and KB articles (KB deferred to 3.8)
8. Suggestions appear within 2 seconds of focusing input
9. Agent can hide/show Echo panel with keyboard shortcut (Cmd+Shift+E)
10. Agent can disable Echo in personal settings if preferred
11. When Echo panel is hidden/collapsed, no AI suggestions are generated (saves tokens)

## Dev Notes

### AI Response Generation Pattern

Follow the pattern established in [echo-conversation-service.ts:585-632](apps/web/src/lib/services/echo-conversation-service.ts#L585-L632) for `generateTextCompletion`:

```typescript
// Dynamic imports for provider SDKs
async function generateTextCompletion(
  prompt: string,
  config: EchoConfig
): Promise<string> {
  const { provider, model, apiKey } = config;

  if (provider === 'openai') {
    const { default: OpenAI } = await import('openai');
    const client = new OpenAI({ apiKey });
    const response = await client.chat.completions.create({
      model,
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.7,
    });
    return response.choices[0]?.message?.content || '';
  } else if (provider === 'anthropic') {
    const { default: Anthropic } = await import('@anthropic-ai/sdk');
    const client = new Anthropic({ apiKey });
    const response = await client.messages.create({
      model,
      max_tokens: 1024,
      messages: [{ role: 'user', content: prompt }],
    });
    return response.content[0]?.type === 'text' ? response.content[0].text : '';
  }
  throw new Error(`Unsupported provider: ${provider}`);
}
```

### Response Suggestion Prompt Template

```
You are Echo, an AI assistant helping a support agent respond to a user's ping.

Conversation History:
{messages formatted as User/Agent/Echo}

Ping Context:
- Status: {status}
- Category: {category}
- Priority: {priority}

Generate a helpful, professional response the agent can use. The response should:
- Directly address the user's most recent message
- Be empathetic but efficient
- Provide actionable next steps if applicable
- Match a professional but friendly tone
- Be ready to send (no placeholders like [NAME])

Write ONLY the response text, nothing else.
```

### Keyboard Shortcut Implementation

Use `useEffect` with keydown listener:
```typescript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.shiftKey && e.key === 'E') {
      e.preventDefault();
      setIsEchoPanelVisible((prev) => !prev);
    }
  };
  window.addEventListener('keydown', handleKeyDown);
  return () => window.removeEventListener('keydown', handleKeyDown);
}, []);
```
