**What We're Building:**
This story integrates real-time KB article suggestions into the "Create Ping" page. As users describe their issue, the system searches the knowledge base using semantic search and displays relevant articles in a sidebar. If an article solves their problem, users can close the ping creation flow without submitting a ticket. This builds on:

- The semantic search infrastructure from Story 4.4 (embeddings + hybrid search)
- The article detail page and modal viewing patterns from Story 4.5
- The existing Create Ping page which already has mock KB suggestions UI scaffolding

**Key Integration Points:**

1. Replace mock KB suggestions with real semantic search API calls
2. Debounced search (500ms) triggered as user types
3. Article modal overlay to view full content without losing draft
4. "This solved my issue" button to close flow and track deflection

## Story

**As a** user,
**I want** to see relevant KB articles while describing my issue,
**so that** I can self-solve without submitting a ticket.

## Acceptance Criteria

1. As user types in "Create Ping" message box, system searches KB in real-time
2. "Related Articles" panel appears below message box if matches found
3. Top 3 articles displayed with title and short preview
4. User can click article to view in modal overlay
5. Modal includes "This solved my issue" button (closes ticket creation flow)
6. If user continues typing after viewing article, ticket creation proceeds normally
7. Debounced search (waits 500ms after typing stops before searching)
8. "View all X results" link expands to full search page

### useDebounce Hook

Existing hook to reuse:

```typescript
// apps/web/src/hooks/use-debounce.ts
import { useState, useEffect } from 'react';

export function useDebounce<T>(value: T, delay: number): T {
  const [debouncedValue, setDebouncedValue] = useState<T>(value);

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => clearTimeout(handler);
  }, [value, delay]);

  return debouncedValue;
}
```

[Source: apps/web/src/hooks/use-debounce.ts]
