**What We're Building:**
This story adds a "Knowledge Base" tab to the existing Analytics dashboard (`/dashboard/analytics`) for managers to understand knowledge base performance. The tab displays popular articles, helpfulness metrics, article creation trends over time, and category breakdowns. Managers can identify high-performing content and gaps in coverage. Export to CSV enables offline analysis and reporting. This builds on:

- The `view_count`, `helpful_count`, and `deflection_count` columns in `kb_articles`
- The `kb_article_views` table tracking individual article views
- The `kb_article_feedback` table for thumbs up/down data
- The `kb_deflections` table from Story 4.6 for self-service tracking
- The existing Analytics page at `/dashboard/analytics` (currently mock data)

**Key Integration Points:**

1. Add "Knowledge Base" tab to existing `/dashboard/analytics` page
2. API endpoints for aggregating KB analytics data with date range filtering
3. Reuse existing chart patterns from the mock analytics page
4. CSV export functionality for KB analytics data

### CSV Export Pattern

Use native JavaScript to generate CSV:

```typescript
function generateCSV(data: Record<string, unknown>[]): string {
  const headers = Object.keys(data[0] || {});
  const rows = data.map(row =>
    headers.map(h => JSON.stringify(row[h] ?? '')).join(',')
  );
  return [headers.join(','), ...rows].join('\n');
}

// In route handler
return new Response(csvContent, {
  headers: {
    'Content-Type': 'text/csv',
    'Content-Disposition': `attachment; filename="kb-analytics-${date}.csv"`,
  },
});
```

### Helpful Ratio Calculation

Calculate helpfulness percentage:

```sql
-- In SQL query
SELECT
  title,
  helpful_count,
  (SELECT COUNT(*) FROM kb_article_feedback WHERE article_id = a.id AND NOT is_helpful) as unhelpful_count,
  CASE
    WHEN helpful_count + unhelpful_count > 0
    THEN ROUND(helpful_count::numeric / (helpful_count + unhelpful_count) * 100, 1)
    ELSE NULL
  END as helpful_percentage
FROM kb_articles a
WHERE status = 'published'
  AND helpful_count + unhelpful_count >= 5  -- minimum votes threshold
ORDER BY helpful_percentage DESC
LIMIT 10;
```

Or in TypeScript:
```typescript
const helpfulPercentage = totalVotes > 0
  ? Math.round((helpfulCount / totalVotes) * 100)
  : null;
```
