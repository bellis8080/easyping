## Story

**As a** manager,
**I want** to see individual agent performance,
**so that** I can identify top performers and coaching opportunities.

## Acceptance Criteria

1. Agent performance table shows: Agent Name, Pings Resolved, Avg Resolution Time, SLA Compliance %, First Response Time
2. Table sortable by each column
3. Agent detail view (click agent name) shows individual metrics and ping history
4. Leaderboard view: Top agents by pings resolved (gamification element)
5. Agent personal dashboard shows their own stats (agents can view their own performance only)
6. Metrics exclude pings created before agent assignment (only measure from assignment time)

#### 1.3 Database Queries (AC: 6)

**CRITICAL:** Metrics must be calculated from assignment time, NOT ping creation time.

- [x] 1.3.1 Get list of agents in tenant:
  ```sql
  SELECT id, full_name, avatar_url FROM users
  WHERE tenant_id = $1 AND role IN ('agent', 'manager', 'owner');
  ```

- [x] 1.3.2 Pings resolved per agent (within date range):
  ```sql
  SELECT assigned_to, COUNT(*) as resolved_count
  FROM pings
  WHERE tenant_id = $1
    AND assigned_to IS NOT NULL
    AND resolved_at IS NOT NULL
    AND resolved_at >= $2 AND resolved_at < $3
  GROUP BY assigned_to;
  ```

- [x] 1.3.3 Avg resolution time per agent (from assignment, not creation):
  ```sql
  SELECT assigned_to,
    AVG(EXTRACT(EPOCH FROM (resolved_at - assigned_at)) / 60) as avg_minutes
  FROM pings
  WHERE tenant_id = $1
    AND assigned_to IS NOT NULL
    AND resolved_at IS NOT NULL
    AND assigned_at IS NOT NULL
    AND resolved_at >= $2 AND resolved_at < $3
  GROUP BY assigned_to;
  ```
  **Note:** Requires `assigned_at` timestamp - may need migration if not exists.

- [x] 1.3.4 SLA compliance per agent:
  ```sql
  SELECT assigned_to,
    COUNT(*) FILTER (WHERE sla_breached = false) * 100.0 /
    NULLIF(COUNT(*), 0) as compliance_rate
  FROM pings
  WHERE tenant_id = $1
    AND assigned_to IS NOT NULL
    AND resolved_at IS NOT NULL
    AND resolved_at >= $2 AND resolved_at < $3
  GROUP BY assigned_to;
  ```

- [x] 1.3.5 Avg first response time per agent:
  ```sql
  SELECT p.assigned_to,
    AVG(EXTRACT(EPOCH FROM (pm.created_at - p.assigned_at)) / 60) as avg_minutes
  FROM pings p
  JOIN ping_messages pm ON pm.ping_id = p.id
  WHERE p.tenant_id = $1
    AND p.assigned_to IS NOT NULL
    AND p.assigned_at IS NOT NULL
    AND pm.sender_id = p.assigned_to
    AND pm.created_at >= $2 AND pm.created_at < $3
    AND pm.created_at = (
      SELECT MIN(pm2.created_at) FROM ping_messages pm2
      WHERE pm2.ping_id = p.id AND pm2.sender_id = p.assigned_to
    )
  GROUP BY p.assigned_to;
  ```

- [x] 1.3.6 Currently assigned pings per agent:
  ```sql
  SELECT assigned_to, COUNT(*) as assigned_count
  FROM pings
  WHERE tenant_id = $1
    AND assigned_to IS NOT NULL
    AND status IN ('new', 'in_progress', 'waiting_on_user')
  GROUP BY assigned_to;
  ```

### Task 2: Add `assigned_at` Timestamp (AC: 6)

**Check if migration needed** - Review existing pings table schema.

**File:** `supabase/migrations/YYYYMMDDHHMMSS_add_assigned_at_to_pings.sql` (if needed)

- [x] 2.1 Check if `assigned_at` column exists in pings table
- [x] 2.2 If not, create migration:
  ```sql
  ALTER TABLE pings ADD COLUMN assigned_at TIMESTAMPTZ;

  -- Backfill: Set assigned_at = created_at for already-assigned pings
  UPDATE pings SET assigned_at = created_at WHERE assigned_to IS NOT NULL AND assigned_at IS NULL;

  -- Create trigger to auto-set assigned_at when assigned_to changes
  CREATE OR REPLACE FUNCTION set_assigned_at()
  RETURNS TRIGGER AS $$
  BEGIN
    IF NEW.assigned_to IS NOT NULL AND OLD.assigned_to IS DISTINCT FROM NEW.assigned_to THEN
      NEW.assigned_at = NOW();
    END IF;
    RETURN NEW;
  END;
  $$ LANGUAGE plpgsql;

  CREATE TRIGGER trigger_set_assigned_at
    BEFORE UPDATE ON pings
    FOR EACH ROW
    EXECUTE FUNCTION set_assigned_at();
  ```

### Time Formatting Helper (reuse from Story 5.4)

```typescript
function formatResolutionTime(minutes: number | null): string {
  if (minutes === null) return 'N/A';
  const hours = Math.floor(minutes / 60);
  const mins = Math.round(minutes % 60);
  if (hours === 0) return `${mins}m`;
  if (mins === 0) return `${hours}h`;
  return `${hours}h ${mins}m`;
}
```
